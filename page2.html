IFfinance

Projeto Desenvolvido em Sala de Aula: Plataforma de Investimentos Este projeto foi criado durante aulas práticas, utilizando as linguagens HTML, CSS e JavaScript para estruturar, estilizar e adicionar interatividade à plataforma. O foco principal do desenvolvimento foi criar uma interface intuitiva e funcional para uma plataforma fictícia de investimentos.
