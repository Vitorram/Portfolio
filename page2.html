Descriçao

O projeto foi desenvolvido em ambiente acadêmico, sob a orientação do professor, com o objetivo de introduzir os conceitos fundamentais das linguagens HTML, CSS e JavaScript. O resultado é um aplicativo web voltado para investimentos, projetado para oferecer uma interface amigável e interativa, permitindo aos usuários simular e acompanhar o desempenho de diferentes cenários de investimento. Durante o desenvolvimento, foram aplicados princípios básicos de estruturação de páginas, estilização e funcionalidade, consolidando o aprendizado prático das tecnologias envolvidas.
